# Faceplate releases

This repository is the public distribution channel for [Faceplate](https://github.com/dxshdw/Faceplate), an iOS music hub sideloaded via LiveContainer. Builds are produced from the private source repo's CI and pushed here automatically on tagged releases.

**LiveContainer source URL:** `https://raw.githubusercontent.com/dxshdw/Faceplate-releases/master/source.json`

The IPAs in `releases/` are unsigned; LiveContainer is required to install them.
